## FASE 1 — M01: Auto-mensajes Conductuales

### Task 2: Cron + Templates de email

**Files to create:**
- `api/cron/behavioral-triggers.php`
- `api/emails/behavioral-templates.php`

**behavioral-triggers.php — logica completa:**

El cron corre diario a las 8am. Para cada cliente activo con email:
1. Calcular `$daysSinceCheckin` = diferencia en dias desde MAX(checkin_date)
2. Calcular `$daysToExpiry` = diferencia desde subscription_end hasta hoy
3. Calcular `$daysSinceJoin` = dias desde created_at
4. Calcular `$totalCheckins` = COUNT(checkins)

Funcion helper `wasSentToday(PDO $db, int $clientId, string $trigger): bool`:
- SELECT de auto_message_log WHERE client_id=? AND trigger_type=? AND DATE(sent_at)=CURDATE()
- Retorna true si ya fue enviado hoy

Funcion helper `logTrigger(PDO $db, int $clientId, string $trigger): void`:
- INSERT IGNORE INTO auto_message_log (client_id, trigger_type, channel) VALUES (?,?,'email')

**8 condiciones evaluadas por cliente:**

| Variable | Condicion | Trigger key | Funcion template |
|----------|-----------|-------------|-----------------|
| $daysSinceCheckin | === 7 | inactive_7d | email_inactive_7d() |
| $daysSinceCheckin | === 14 | inactive_14d | email_inactive_14d() |
| $daysToExpiry | === 7 | subscription_7d | email_renewal_reminder($days=7) |
| $daysToExpiry | === 3 | subscription_3d | email_renewal_reminder($days=3) |
| $totalCheckins | === 4 | streak_4 | email_streak_milestone($weeks=4) |
| $totalCheckins | === 7 | streak_7 | email_streak_milestone($weeks=7) |
| birthday MM-DD | === hoy MM-DD | birthday | email_birthday() |
| $daysSinceJoin | === 1 | welcome_day1 | email_welcome_day1() |

Query principal del cron:
```sql
SELECT c.id, c.name, c.email, c.plan, c.subscription_end, c.birth_date, c.created_at,
       MAX(ch.checkin_date) AS last_checkin,
       COUNT(ch.id) AS total_checkins
FROM clients c
LEFT JOIN checkins ch ON ch.client_id = c.id
WHERE c.status = 'activo' AND c.email != ''
GROUP BY c.id
```

```javascript
var VAPID_PUB_KEY = 'REEMPLAZAR_CON_VAPID_PUBLIC_KEY_DEL_ENV';

function enablePushNotifications() {
  if (!('serviceWorker' in navigator) || !('PushManager' in window)) {
    showToast('Push no soportado en este navegador', 'error'); return;
  }
  Notification.requestPermission().then(function(perm) {
    if (perm !== 'granted') { showToast('Permiso denegado', 'error'); return; }
    navigator.serviceWorker.ready.then(function(reg) {
      reg.pushManager.subscribe({
        userVisibleOnly: true,
        applicationServerKey: urlBase64ToUint8Array(VAPID_PUB_KEY)
      }).then(function(sub) {
        apiFetch('/push/subscribe', { method: 'POST', body: JSON.stringify(sub.toJSON()) })
          .then(function() { showToast('Notificaciones activadas', 'success'); });
      }).catch(function(e) { showToast('Error: ' + e.message, 'error'); });
    });
  });
}

function urlBase64ToUint8Array(base64String) {
  var padding = '='.repeat((4 - base64String.length % 4) % 4);
  var base64  = (base64String + padding).replace(/-/g, '+').replace(/_/g, '/');
  var rawData = atob(base64);
  var output  = new Uint8Array(rawData.length);
  for (var i = 0; i < rawData.length; ++i) output[i] = rawData.charCodeAt(i);
  return output;
}
```
