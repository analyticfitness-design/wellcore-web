Agregar Clientes

Los clientes se gestionan via la base de datos MySQL. Usa el panel Admin Clientes para crear, editar o invitar clientes. Tambien puedes usar el sistema de Patrocinio desde Admin Jefe.

POST /api/admin/create-sponsored
POST /api/admin/send-invitation

Los clientes tambien se crean automaticamente al completar inscripcion o pago via Wompi.

@

Formspree

Formulario de contacto configurado via Formspree. Reemplaza el ID en el action del formulario principal.

action="https://formspree.io/f/TU_FORM_ID"
→ formspree.io →
◈

Wompi (Bancolombia)

Pasarela de pagos. Acepta tarjetas, PSE, Nequi, Efecty. Configura las llaves en api/.env.

WOMPI_PUBLIC_KEY=pub_...
WOMPI_PRIVATE_KEY=prv_...
WOMPI_INTEGRITY_SECRET=prod_integrity_...
→ comercios.wompi.co →
